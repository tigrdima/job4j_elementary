  -4.2. Заголовок в файле java

  В этом уроке я познакомился  с элементом "Package".

  Имя файла должно быть уникальным.
  Чтобы соблюдать это правило самостоятельно, нам бы
  приходилось именовать классы с использованием номеров.
  Например, User1 User100500. Такое именование привело
  бы к хаосу.

  Чтобы этого избежать, в Java необходимо использовать
   полное имя класса.

  Это имя состоит из пути от папки "src" до файла *.java.
